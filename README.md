# README
## 開発言語
* ruby 3.0.1
* Rails 6.1.7.6
## 就職Termの技術
* gem devise
* AWS EC2
## カリキュラム外の技術
* gem ransack
## 実行手順
```
$ git clone git@github.com:dejima-yuya/doctor-agri.git
$ cd doctor-agri
$ bundle install
$ yarn install
$ rails db:create && rails db:migrate
$ rails db:seed
$ rails s
```
